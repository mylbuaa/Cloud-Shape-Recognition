# Cloud-Shape-Recognition
#1 Introduction
细粒度识别是计算机视觉的一个不断发展的子领域，在识别细微的外观和差异方面有着许多实际应用。该数据集包含28种云状类型，你需要根据图像进行细粒度分类，类别包括：中云-高积云-絮状高积云、中云-高积云-透光高积 云、高云-卷云-伪卷云、低云-雨层云-雨层云等。
